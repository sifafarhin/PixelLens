# PixelLens – AI UI/UX Analyzer

PixelLens is an AI-powered web application that analyzes website screenshots and provides UI/UX improvement recommendations.  
It helps developers and designers improve the visual design, usability, and overall user experience of their websites using AI.



# Features

- Upload website screenshots for analysis
- AI-generated UI/UX improvement suggestions
- Clean and simple user interface
- Fast and responsive design
- Easy-to-use workflow for designers and developers



# Tech Stack

- React
- Node.js
- Express.js
- Tailwind CSS
- Gemini AI



# How It Works

1. User uploads a website screenshot  
2. Backend sends the image to Gemini AI  
3. AI analyzes UI/UX design  
4. Suggestions are returned to the user in a readable format
